## Server Requirements
* PHP version 5.3.0 or greater (But recommend PHP >= 5.5.0)
* MYSQL version 4.1 or greater (But recommend MYSQL >= 5.x) , MS-SQL, CUBRID
* XML Library
* GD Library
* Fileinfo Extension
* ICONV
* session.auto_start = Off (php.ini)
